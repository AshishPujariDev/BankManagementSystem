# Bank Management System

## Project Description
Bank Management System is a Core Java console-based application developed using Java, JDBC, Maven, and MySQL. It allows users to manage customer bank accounts efficiently.

## Technologies Used
- Java
- JDBC
- MySQL
- Maven
- IntelliJ IDEA
- Git
- GitHub

## Features/
- Create Account
- View All Customers
- Deposit Money
- Withdraw Money
- Check Balance
- Delete Account
- Search Customer
- Update Customer

## Transaction History

The system stores every Deposit and Withdraw transaction in the database.

Each transaction contains:
- Account Number
- Transaction Type (Deposit/Withdraw)
- Amount
- Date & Time

Users can view complete transaction history from the menu.
